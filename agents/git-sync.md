// turbo-all

Этот воркфлоу позволяет мне автоматически собрать все измененные файлы, сделать коммит со сгенерированным описанием (или дефолтным) и отправить их в ветку текущего репозитория.

Шаги, которые я выполню без запроса разрешений:
1. `git status` (чтобы понять, что изменилось)
2. `git add .` (добавить все файлы)
3. `git commit -m "Автоматическое сохранение изменений"` (сделать коммит)
4. `git push` (отправить на сервер)
